# Overview

利用**本地播放器**播放战旗游戏娱乐室的直播

### News

+新增了B站直播

### Requirements

#### For Linux

```
$ sudo apt install python3
$ sudo apt install mpv
$ [sudo] pip3 install requests
```
#### For Windows

- python3
- Potplayer播放器
- pip3 install requests

### Usage

`python3 ./main.py` for linux
`python ./main.py` for Windows
